# 📊 Customer Behavior Analysis Dashboard (Power BI)

## 🔍 Overview

This project presents an interactive **Customer Behavior Dashboard** built using Power BI.
It focuses on analyzing customer purchasing patterns, subscription trends, and sales performance to generate meaningful business insights.

---

## 📁 Dataset

The dataset contains customer-related information such as:

* Customer ID
* Purchase Amount
* Product Category
* Subscription Status
* Age Group
* Review Ratings
* Shipping Preferences

---

## ⚙️ Tools & Technologies

* Power BI
* Data Visualization
* Data Cleaning & Transformation
* Dashboard Design

---

## 📈 Key Insights

* Total Customers: **324**
* Average Purchase Amount: **$57.17**
* Average Review Rating: **3.75**
* Majority of customers are **non-subscribers**
* Highest revenue generated from **Outerwear category**
* Senior and middle-aged groups contribute more to revenue

---

## 📊 Dashboard Features

* KPI Cards (Customers, Avg Purchase, Rating)
* Subscription Analysis (Donut Chart)
* Revenue & Sales by Category
* Age Group Analysis
* Interactive Filters (Category, Shipping, Subscription)

---

## 🎯 Objectives

* Understand customer purchasing behavior
* Identify high-performing product categories
* Analyze customer segments based on age and subscription
* Support data-driven decision making

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Explore the dashboard using filters and visuals
